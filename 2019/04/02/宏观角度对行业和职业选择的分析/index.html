宏观角度对行业和职业选择的分析

发表于 2019-04-02 | 分类于 思考 |

商业行为的本质是交换，为了促进交换，发展出了形形色色的商业组织。货币的产生也是为了更好地促进交换。而之所以发生交换是因为有需求，因此可以说需求是商业行为发生的本质。

需求是在经济社会中所有变化的核心，需求不变经济结构就不会发生变化。我们沿着需求这个脉络就可以很好地把握未来职业选择。

在具体分析之前，需要两个先验的假设。

第一个假设是中国经济会持续中高速发展，未来十年国际政治经济环境不会发生剧烈动荡变化。中国政府可以完成消费升级的重任，培育一大批新中产阶级。

第二个假设是数字经济会持续推进，深刻影响并改造各行各业。

基于上面两个假设，下面从宏观层面对行业选择进行一下分析。

中国目前人均GDP在9000美元左右，未来中国经济还会持续增长。当前，党和政府越来越注重依靠消费来拉动经济增长，要靠消费来拉动经济增长，就一定要培育一大批中产阶级。

在改革开放的40年中，中国经济增长最快的时候是加入WTO的过去20年里。相应的，整体来看，中国最有钱的人也应该分布在这一年龄段。

这一年龄段的人，还没有受到全面实施计划生育政策的影响。但是，随着计划生育政策全面展开，中国人口结构发生了很大变化。

全世界都存在人口老龄化的问题，但这一问题在中国会尤其严重。

《国家应对人口老龄化战略研究总报告》中将中国人口老龄化分为4个阶段。[^1]

  1. 快速老龄化阶段（1999-2022年）

    老龄人口数量从1.31亿增至2.68 亿，老龄化水平从10.3%升至18.5%。

    2009年，随着1949-1958年第一次出生高峰人口迸入老齢，中国老齢人口迸入第一个増长高峰，年均浄増加840万，年均増长率4.07%，是同期总人口年均増七率的6倍。

    2022年，随着1962-1976年第二次出生高峰人口迸入老龄，中国老龄人口迎来第二个増长高峰，年均浄増加1106万，年均増长率3.26%，是同期总人口年均増长率的98倍。中国老龄人ロ与少年儿童人口数量相同。之后，老龄人口数量幵始超过少儿人口数量，中国进入以抚养老龄人口为主的时代。

  2. 急速老龄化阶段（2023-2036年）

    老龄人口数量从2.68亿增至4.23亿，老龄水平从18.5%升至29.1%。

    2029年，中国总人口数量达到峰值14.63亿；2023年，中国总人口开始负增长。

  3. 深度老龄化阶段（2036-2053年）

    老龄人口数量从4.23亿达到4.87亿的峰值，人口老龄化水平从29.1%升至34.8%。

    2046年，随着1987-1995年第三次出生高峰人口进入老龄，中国人口老龄人口迎来第三个增长高峰，年均增加666万，年均增长率1.42%。

  4. 重度老龄化平台阶段（2054-2100年）
    老龄人口增长期结束，由4.87亿减少到3.83亿，人口老龄化水平始终稳定在34%上下。

可以看到在2022年左右的人口老龄化，刚好对应的是在改革开放中获利最大一部分人口群体。他们基本上也没有什么房贷压力，有比较高的收入。对于一个人来说，基本物质需求是有上限的，当一个人有了很多钱的时候，基本生活花销需求，远远比不上对于精神娱乐、投资理财、教育、旅游、健康、养老这样的服务业。

健康和养老保健等是一个很大的增量市场，无论是现在996.ICU的程序员，还是看不起病的穷人，亦或者是长期忍受慢性病痛苦折磨的人，能够让他们少生病，以及防患于未然，我觉得是一件很有意义的事情。

第二个是教育市场，我能做的只能是在线教育。根据《中国互联网发展状况统计报告》（2019年3月）[^2]

可以看到目前整个中国网民有低学历以及学生居多的特点。

可以看到职业教育和成人教育占据教育市场大部分份额。成人教育大规模发展，以及以学生为主的年轻网民，将技能通过在线教育的方式变现，是一个很好的副业。特别是成人在线教育依旧野蛮生长，监管法规还不算健全的时候。

最后说一点对于数字经济的展望。接下来的物联网时代并不是人唱主角的时代，大规模的物物相连，并用人工智能等方式赋予其智能。这就会造就许许多多的印钞机。比如现在的百度搜索，搜索内容是算法自动找到的，旁边的广告也是算法自动推荐的，人们只是设计和维护这个系统，系统开起来之后就可以源源不断的造钱。那么，我们设想一下，如果将来街上跑的都是滴滴的无人驾驶出租车，农田里，无人操控的无人机自主的喷洒农药，人们的劳动会被大大的减轻了。对于资本家来说，他可以购买一个又一个这样类似的印钞机，那么创造的财富很大一部分会归这些资本家所有，财富会进一步的高度集中。当然，这可能是未来20年以后的事情。

如果这个事情真的被我言中，不幸发生的话，从现在开始学习投资理财知识是很有必要的。
